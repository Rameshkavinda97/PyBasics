Variables & Data Types

Learn how to store and manage data in your programs

What is a Variable?

Think of a variable as a labeled container that holds information. Just like you might have a box labeled "Photos" to store pictures, a variable stores data that your program can use and change.

In Python, you create a variable by giving it a name and assigning a value using the equals sign (=).

Python variables.py 
# Creating variables - storing different types of data

# String (text) - wrapped in quotes
name = "Alice"
greeting = 'Hello, World!'

# Integer (whole number) - no quotes needed
age = 25
year = 2026

# Float (decimal number)
price = 19.99
temperature = 36.6

# Boolean (True or False)
is_student = True
has_license = False

# Printing variables to see their values
print(name)        # Output: Alice
print(age)         # Output: 25
print(is_student)  # Output: True

üíª Variables in Action

Variables are the building blocks of every program. They allow your code to store, retrieve, and manipulate data dynamically. As you progress, you'll see variables in every program you write!

üí° Variable Naming Rules

  • Use descriptive names: user_age is better than x
  • Start with a letter or underscore, not a number
  • Use lowercase with underscores: first_name (snake_case)
  • Avoid Python keywords like print, if, for

Python Data Types

Here are the four basic data types you'll use most often:

üìù String (str)

Text data wrapped in quotes. Can use single or double quotes.

"Hello" or 'World'

üî¢ Integer (int)

Whole numbers without decimals. Can be positive or negative.

42, -10, 0

üìä Float (float)

Numbers with decimal points for precision.

3.14, -0.5, 100.0

‚úÖ Boolean (bool)

Only two values: True or False. Used for decisions.

True, False

Loops & Iteration

Repeat actions efficiently without writing redundant code

Why Use Loops?

Imagine you need to print "Hello" 100 times. Without loops, you'd have to write print("Hello") 100 times! Loops let you repeat code automatically ‚Äî saving time and making your code cleaner.

The For Loop

A for loop repeats code for each item in a sequence (like a list or range of numbers). It's perfect when you know how many times you want to repeat.

Python for_loop.py 
# For loop with range() - repeat 5 times
for i in range(5):
    print(f"Count: {i}")

# Output:
# Count: 0
# Count: 1
# Count: 2
# Count: 3
# Count: 4

# For loop with a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I love {fruit}!")

# Output:
# I love apple!
# I love banana!
# I love cherry!

The While Loop

A while loop repeats code as long as a condition is true. It's useful when you don't know exactly how many times you need to repeat.

Python while_loop.py 
# While loop - repeat until condition is false
count = 0

while count < 3:
    print(f"Count is: {count}")
    count = count + 1  # Important: update the counter!

print("Loop finished!")

# Output:
# Count is: 0
# Count is: 1
# Count is: 2
# Loop finished!

‚ö†Ô∏è Watch Out: Infinite Loops

If the condition in a while loop never becomes False, your program will run forever! Always make sure to update variables inside the loop so it can eventually stop.

Conditionals (If/Else)

Make your programs smart with decision-making

Making Decisions in Code

Just like you make decisions every day ("If it's raining, I'll take an umbrella"), programs use conditionals to choose different actions based on certain conditions.

Python conditionals.py 
# Basic if statement
age = 18

if age >= 18:
    print("You are an adult!")

# If-else statement
temperature = 25

if temperature > 30:
    print("It's hot outside!")
else:
    print("The weather is nice.")

# If-elif-else for multiple conditions
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"Your grade is: {grade}")  # Output: Your grade is: B

üí° Indentation Matters!

Python uses indentation (spaces at the beginning of lines) to group code. The code inside an if statement must be indented (usually 4 spaces). This is what makes Python unique and readable!

Functions

Organize your code into reusable building blocks

What is a Function?

A function is a reusable block of code that performs a specific task. Instead of writing the same code over and over, you can define it once in a function and call it whenever you need it.

Think of it like a recipe: you write the instructions once, then follow them whenever you want to make that dish.

Python functions.py 
# Defining a simple function
def say_hello():
    """This function prints a greeting."""
    print("Hello, World!")

# Calling the function
say_hello()  # Output: Hello, World!
say_hello()  # You can call it multiple times

# Function with a parameter
def greet(name):
    """Greet a person by name."""
    print(f"Hello, {name}!")

greet("Alice")   # Output: Hello, Alice!
greet("Bob")     # Output: Hello, Bob!

Return Values

Functions can return a value back to where they were called. This is useful when you need to use the result of a calculation.

Python return_values.py 
# Function that returns a value
def add_numbers(a, b):
    """Add two numbers and return the result."""
    result = a + b
    return result

# Using the returned value
total = add_numbers(5, 3)
print(f"5 + 3 = {total}")  # Output: 5 + 3 = 8

# You can use the result directly
print(add_numbers(10, 20))  # Output: 30

# A more practical example
def calculate_area(width, height):
    """Calculate the area of a rectangle."""
    return width * height

room_area = calculate_area(10, 12)
print(f"Room area: {room_area} square meters")

üí° Function Best Practices

  • Give functions descriptive names that explain what they do
  • Keep functions focused on one task
  • Add a docstring (description in triple quotes) to explain the function
  • Use parameters to make functions flexible
