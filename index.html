Your First Python Code

Here's a taste of what you'll be writing. Python is known for its clean, readable syntax.

Python hello_world.py 
# This is your first Python program!
# It prints a friendly greeting to the screen

def greet(name):
    """Say hello to someone."""
    message = f"Hello, {name}! Welcome to Python."
    return message

# Let's use our function
greeting = greet("Beginner")
print(greeting)

# Output: Hello, Beginner! Welcome to Python.
