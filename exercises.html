Practice creating and using variables

Easy

Exercise 1: Personal Info Variables

Create variables to store your personal information:

  • Your age (integer)
  • Your height in meters (float)

Exercise 8: Age Checker

Create a variable for age. Write an if-else statement that:

Write a program that takes a score (0-100) and prints the corresponding grade:

Exercise 10: Leap Year Checker

Write a program that checks if a year is a leap year. A year is a leap year if:

  • BUT NOT divisible by 100

üí° Hint

Exercise 13: Temperature Converter

Create two functions:

  • celsius_to_fahrenheit(celsius) ‚Äî converts and returns Fahrenheit
  • fahrenheit_to_celsius(fahrenheit) ‚Äî converts and returns Celsius

Formulas: F = C √ó 9/5 + 32 and C = (F - 32) √ó 5/9

Test Cases 
# Test your functions with these:
print(celsius_to_fahrenheit(0))     # Should print: 32

print(fahrenheit_to_celsius(212))   # Should print: 100

üö´ Common Errors to Avoid

Watch out for these common beginner mistakes:

IndentationError

Python requires consistent spacing. Use 4 spaces for each indentation level.

SyntaxError

Missing colons after if, for, while, or def statements.

NameError

Using a variable before defining it, or misspelling a variable name.

TypeError

Trying to combine incompatible types (like adding a string to a number).
